# Dify AI Workflow Developer Resources

## Knowledge

- [Official Dify Documentation](https://docs.dify.ai/)
  The definitive guide for understanding Dify. Use for: learning about Workflow nodes (LLM, Code, Template, Knowledge), API integrations, and system architecture.
- [Dify Workflow Introduction](https://docs.dify.ai/features/workflow)
  Detailed guide on how to design and build workflows. Use for: studying workflow components, variables, and control flow.
- [Dify Code Node Guide](https://docs.dify.ai/features/workflow/nodes/code)
  Documentation on using Javascript/Python nodes inside workflows. Use for: syntax restrictions, sandbox environments, and input/output schema definitions.

## Wisdom (Communities)

- [Dify Discord Server](https://discord.gg/FngnHQ3jDB)
  The official community channel. Use for: sharing custom templates, troubleshooting pipeline errors, and discussing best practices with other workflow builders.
- [Dify GitHub Issues & Discussions](https://github.com/langgenius/dify/discussions)
  Open-source community forum. Use for: reporting bugs, proposing new features, and seeing how others deploy Dify self-hosted.
