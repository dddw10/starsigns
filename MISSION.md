# Mission: 使用 Dify 搭建前端/软件开发自动化工作流

## Why
作为前端开发者，通过结合 Dify 平台与已有的 JavaScript 能力，搭建与软件开发深度集成的自动化工作流。通过 AI 自动化处理日常开发中的重复琐碎事务（如接口代码生成、文档编写、自动化测试生成等），显著提升个人与团队的研发交付效率。

## Success looks like
- **搭建端到端工作流**：在 Dify 中成功跑通并上线至少一个实用的开发自动化流（如：输入 JSON 自动生成 TypeScript Interface 与 API 请求函数）。
- **掌握 JS 节点高级应用**：在 Dify 的 Workflow 中熟练编写 JavaScript 节点进行数据清洗、格式转换与逻辑分流。
- **本地/工具链对接**：将 Dify API 成功对接本地开发工具或 Webhook，实现无缝的工作流调用。

## Constraints
- **技术背景**：已具备熟练的前端开发（HTML/JS）经验，无需重复学习基础代码知识，重点在于如何将代码与 AI 节点进行组装和逻辑编排。
- **学习平台**：以 Dify 平台（自建或云端版）为主。

## Out of scope
- 大语言模型的底层预训练、微调（Fine-tuning）与复杂的 Transformer 架构原理。
- 零代码通用办公流程（如自动生成 PPT、财务报表等非开发场景自动化）。
- 复杂的基础设施运维与高并发架构（重点在逻辑跑通与业务实用）。
